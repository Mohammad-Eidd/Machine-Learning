📌 Dataset Summary

The selected dataset contains customer behavioral data from an online business.
It includes 1200 records and 4 numerical features:

avg_purchase_value – average amount spent per purchase

purchase_frequency – number of purchases made by the customer

website_visits_per_month – number of monthly website visits

returns_rate – percentage of returned products

The dataset is in CSV format and is suitable for machine learning analysis.

🎯 Machine Learning Problem

This project addresses a Clustering problem (Unsupervised Learning).

There is no target variable in this dataset.
The objective is to group customers into meaningful segments based on their purchasing behavior and activity patterns.

The model is expected to discover natural groupings such as:

High-value customers

Frequent shoppers

Customers with high return rates

Less active customers

Customer segmentation can help businesses improve marketing strategies and customer targeting.
